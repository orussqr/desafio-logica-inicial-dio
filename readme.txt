O script classifica o nível de um herói com base na quantidade de XP (experiência) informada.

A lógica utiliza estruturas condicionais (if / else if / else) 
para determinar em qual faixa de nível o herói se encontra.
______________________________________________________________________________________________

Foi definido duas variáveis:

nick: nome do herói
xp: quantidade de experiência do herói

______________________________________________________________________________________________

Como executar:

Instale o Node.js
Salve o arquivo como index.js
Execute no terminal: node index.js
